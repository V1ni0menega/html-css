Guia Completo: Propriedades e Interações CSS

Veja abaixo uma lista abrangente das propriedades CSS mais comuns e suas funções:

PropriedadeFunção
Texto
colorDefine a cor do texto
font-sizeTamanho da fonte
font-familyFonte do texto
font-weightNegrito (normal, bold, lighter)
font-styleEstilo (normal, italic)
text-alignAlinhamento do texto (left, right, center, justify)
text-decorationSublinhado, linha sobre o texto etc.
line-heightAltura da linha
letter-spacingEspaçamento entre letras
word-spacingEspaçamento entre palavras
text-shadowSombra no texto
Caixa (Box Model)
marginMargem externa
paddingEspaço interno
borderBorda (largura, estilo, cor)
width / heightLargura e altura
box-shadowSombra da caixa
overflowComportamento do conteúdo que excede
Layout
displayTipo de exibição (block, inline, flex, grid...)
positionPosição (relative, absolute, fixed, sticky)
top, right, bottom, leftDeslocamento com position
z-indexOrdem de sobreposição
floatFlutuar ao lado (esquerda, direita)
clearLimpar floats
flex, justify-content, align-itemsAlinhamento com Flexbox
grid, grid-templateDefinição de grades com Grid
Visual
background-colorCor de fundo
background-imageImagem de fundo
opacityTransparência
filterFiltros visuais (blur, brightness...)
border-radiusBordas arredondadas
visibilityVisibilidade (visible, hidden)
cursorTipo de ponteiro (ex: pointer)
Animação e Efeitos
transitionTransição suave entre estados
transformTransformações (girar, mover, escalar...)
animationDefinir animações com keyframes
:hover, :active, :focusEstados de interação do usuário
@mediaMedia queries para responsividade
